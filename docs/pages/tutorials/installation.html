Installing ardUI

Installing ardUI was made to be as easy as possible. To start using ardUI just download the latest version of the framework from the repository. If you are using the Arduino IDE, then you can import the archive directly (here is how you can do it). If you are using some other environment, like PlatformIO, then extract all files from archive to your library directory, which would depend on your framework. To test whether ardUI is installed correctly, add #include "ardUI.h" and compile your project. If you get no errors, it means that you are ready to start using ardUI!
